# post-your-work-project
Posting My Work Project Using GitHub
## Information about how to use your project
Cloned the repository from GitHub to the local machine
Added our bikeshare project files into the local repository
Created a .gitignore file (bikeshare_2.gitignore) to exclude data files from being tracked
Staged the changes to prepare them for committing
Committed the changes with descriptive messages
Pushed the commits to the remote GitHub repository
